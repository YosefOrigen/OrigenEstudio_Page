Introducción

Un motor de videojuego es el conjunto de herramientas, sistemas y recursos que hacen posible construir, renderizar y ejecutar una experiencia interactiva. Incluye física, audio, animaciones, renderizado y gestión de escenas.

¿Motor vs. Código?

El motor ofrece la infraestructura técnica base; el código define la lógica, la jugabilidad y los comportamientos personalizados. Juntos permiten iterar más rápido y centrar el desarrollo en la experiencia del jugador.

¿Por dónde empezar?

Empieza por comprender los componentes principales del motor y su flujo de trabajo. Después, crea proyectos pequeños para aplicar conceptos y elaborar tu propia metodología de desarrollo.

Código: HTML del menú lateral

El menú lateral utiliza una estructura semántica con aside, botones de acordeón y enlaces de navegación interna.

<aside class="sidebar">
  <nav class="sidebar-nav">
    <div class="accordion-section">
      <button class="accordion-toggle">Introducción</button>
      <ul class="accordion-content">
        <li><a href="#que-es-un-motor">¿Qué es un motor de videojuego?</a></li>
      </ul>
    </div>
  </nav>
</aside>

CSS para sidebar fijo

El estilo aplica position: sticky para que el menú permanezca visible al desplazarse, junto con un diseño moderno y legible.

.sidebar {
  position: sticky;
  top: 1.5rem;
  max-height: calc(100vh - 3rem);
  overflow: auto;
  padding: 24px;
  background: rgba(255, 255, 255, 0.95);
  border: 1px solid rgba(20, 20, 20, 0.08);
  border-radius: 28px;
}

JS para acordeón

La función alterna el panel oculto o visible y ajusta el max-height para una transición suave.

document.querySelectorAll('.accordion-toggle').forEach((toggle) => {
  toggle.addEventListener('click', () => {
    const panel = toggle.nextElementSibling;
    const opened = panel.classList.toggle('open');
    toggle.setAttribute('aria-expanded', opened);
    toggle.classList.toggle('active', opened);
    panel.style.maxHeight = opened ? panel.scrollHeight + 'px' : null;
  });
});
